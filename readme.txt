# Симулятор полета ракеты 🚀

**Программа для физического моделирования и визуализации траектории полета ракеты (или любого тела) с учетом сопротивления воздуха.**

## Возможности

- 🎯 **Расчет траектории** с учетом гравитации и аэродинамического сопротивления
- 📊 **Два метода моделирования**: аналитический (без сопротивления) и численный (с сопротивлением)
- 🖥️ **Визуализация в консоли** – ASCII-график траектории
- 🎮 **Интерактивная WPF-анимация** с элементами управления:
  - Play/Pause анимации
  - Ползунок прогресса
  - Ускорение в 10 раз
  - Поворот ракеты по направлению движения
  - След траектории
- 📈 **Окно с результатами** полета (время, дальность, высота)

## Входные параметры

| Параметр | Описание |
|----------|----------|
| Угол броска | 0–90 градусов |
| Время | Максимальное время расчета (сек) |
| Начальная скорость | м/с |
| Начальная высота | метры |
| Коэффициент сопротивления | 0–1.0 |
| Масса | кг |
| Площадь сечения | м² |

## Технологии

- C# .NET
- WPF (Windows Presentation Foundation)
- Численное интегрирование (метод Эйлера)

## Пример запуска

Программа открывает диалоговое окно для ввода параметров, выполняет расчет и показывает:
1. Консоль с таблицей и ASCII-графиком
2. Окно с результатами
3. Анимированное окно полета
